Welcome to Teacher Assistant

(The one stop application to automate all teacher related clerical task)

Teacher Assistant is designed to make a Teachers life easy. The idea behind Teacher Assistant is to automate most of the clerical task in a teachers work routine, so that he/she can spend more collective time to the most important part of work i.e. teaching.

Currently Teacher Assistant provides the following three functionalities.

i) Time Table Generation

ii) Report Card Management

iii) Student Profile Management
